Add animated host files here.

Required filenames:

- sent.webp or sent.gif - host walks in and checks his watch
- accepted.webp or accepted.gif - host looks happy and waits
- preparing.webp or preparing.gif - host waits while kitchen cooks
- ready.webp or ready.gif - host presents food / points customer to restaurant
- completed.webp or completed.gif - host thanks the customer
- rejected.webp or rejected.gif - host apologizes and gestures to call restaurant

Animated WebP is preferred because it is smaller, smoother, and supports transparency better than GIF. The site tries WebP first and falls back to GIF.
