Daily workflow:

1. Open the staff dashboard and enter the password.
2. Choose a branch from the branch filter, or use All branches to monitor everything.
3. For a new order, set prep minutes and press Accept.
4. Use Mark Preparing, Mark Ready, Complete, or Reject as the kitchen progresses.
5. Use Kitchen Mode during busy service for larger tap-friendly order cards.
6. Use Print Ticket for kitchen receipts.
7. Use Print Today for a daily report filtered to the selected branch.

Branch availability:

1. Open Menu Availability.
2. Select one branch first. All branches is view-only.
3. Set a dish to Available, Sold out, or Unavailable for now.
4. Use quick buttons like Until lunch or Until 18:00 when useful.
5. Press Save. The change affects only the selected branch.
